# Marketing ROI Analysis

## Overview
Proyecto de análisis financiero con SQL para evaluar ROI, margen y rentabilidad por país. Detecté mercados con mayor potencial de escalamiento y otros de bajo retorno, generando recomendaciones para optimizar inversión publicitaria y mejorar la toma de decisiones de negocio.

## Objective
Evaluar el desempeño financiero y de marketing por país para identificar mercados más rentables y oportunidades de optimización presupuestaria.

## Tools
- SQL
- ROI Analysis
- Financial Analysis
- KPI Analysis
- Excel

## Process
- Integración de datos de ventas, costos y marketing
- Cálculo de ROI y margen por país
- Comparación del desempeño entre mercados
- Identificación de oportunidades de optimización
- Resumen ejecutivo de hallazgos y recomendaciones

## Key Findings
- Se identificaron mercados con mayor retorno sobre inversión
- ROI y margen no siempre presentaron el mismo comportamiento
- Algunos mercados mostraron oportunidades claras de optimización de presupuesto

## Business Value
Este análisis permite tomar decisiones más inteligentes sobre asignación de inversión, expansión comercial y eficiencia de marketing.
